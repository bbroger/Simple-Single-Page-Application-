A simple way to run small spa without needing a framework

  • First define your main div that will cover all the content of your website

    const contentEl = document.getElementById("content");

  • Then define your routes in an object

    const routes ={ home: { path: './views/home.html' }, about: { path: './views/about.html' }, contact: { path: './views/contact.html' } }

  • Then create a new instance of Router and pass the following parameters

    const router = new Router({
      contentEl: contentEl,
      routes: routes,
      linksEl: linksEl
    })

  • Then init the Router

    router.init()

  • Add an active class to the link setting the class name to the attribute active-class

    <a href="#home" active-class="active">
